================================================================================
                          HOW TO GET STARTED
================================================================================

STEP 1: START BACKEND (Terminal 1)
─────────────────────────────────
  cd Backend
  
  Create .env file with:
    DATABASE_URL=postgresql://user:password@localhost:5432/qcm_study_db
    SECRET_KEY=your-secret-key-minimum-32-characters-long
    ALGORITHM=HS256
    ACCESS_TOKEN_EXPIRE_MINUTES=30
    REFRESH_TOKEN_EXPIRE_DAYS=7
    CORS_ORIGINS=http://localhost:3000
  
  Then run:
    pip install -r requirements.txt
    python init_db.py
    uvicorn app.main:app --reload --port 8000
  
  Expected: "Uvicorn running on http://0.0.0.0:8000"

STEP 2: START FRONTEND (Terminal 2)
───────────────────────────────────
  cd Frontend
  pnpm install
  pnpm dev
  
  Expected: "Local: http://localhost:3000"

STEP 3: TEST
────────────
  1. Open http://localhost:3000/inscription
  2. Fill out 3-step registration form
  3. Click "Créer mon compte" (Create account)
  4. Should redirect to confirmation page
  5. Open http://localhost:3000/connexion
  6. Enter email and password from registration
  7. Click "Se connecter" (Sign in)
  8. Should redirect to dashboard!

──────────────

export default function Profile() {
  const { user, isLoading } = useCurrentUser()
  
  return <div>Welcome {user?.first_name}</div>
}

ADMIN ONLY:
───────────
"use client"
import { useRequireAdmin } from "@/lib/auth-hooks"

export default function AdminPage() {
  const { isAdmin, isLoading } = useRequireAdmin()
  
  if (!isAdmin) return <div>Access Denied</div>
  return <div>Admin Content</div>
}

================================================================================
                          TROUBLESHOOTING
================================================================================

BACKEND WON'T START:
  • Check PostgreSQL is running
  • Check DATABASE_URL is correct
  • Check SECRET_KEY is set (min 32 chars)
  • Check port 8000 is not in use

FRONTEND WON'T START:
  • Check Node.js is installed
  • Run: pnpm install (not npm)
  • Check .env.local exists
  • Check port 3000 is not in use

REGISTRATION FAILS:
  • Check backend is running on 8000
  • Check .env.local API_ENDPOINT is correct
  • Check browser console for errors
  • Check Network tab for 401/404/500 errors

LOGIN FAILS:
  • Check email is registered
  • Check password is correct
  • Check backend is running
  • Check browser localStorage has tokens

TOKENS NOT SAVING:
  • Check localStorage permissions
  • Check browser privacy mode is off
  • Clear localStorage and retry
  • Check token key names in .env.local

DASHBOARD NOT LOADING:
  • Check /tableau-de-bord and /admin-dashboard pages exist
  • Check user has correct rank in database
  • Check tokens in localStorage are valid
  • Check browser console for errors
