## рџ”„ How It Works

```
User Form (3-step) 
  в†’ Client Validation 
  в†’ POST /api/v1/auth/register 
  в†’ Backend Creates User 
  в†’ Returns JWT Tokens 
  в†’ Store in localStorage 
  в†’ Redirect to Confirmation вњ…

### Login Flow:
```

  в†’ Client Validation 
  в†’ POST /api/v1/auth/login

  в†’ Returns JWT Tokens 
  в†’ GET /api/v1/auth/me

## рџЋЇ Code Examples

### Protect a Route:
```tsx
"use client"
import { useProtectedRoute } from "@/lib/auth-hooks"

export default function Dashboard() {
  const isAuth = useProtectedRoute()
  
  if (isAuth === null) return <div>Checking...</div>
  return <div>Dashboard Content</div>
}
```

### Get User Info:
```tsx
"use client"
import { useCurrentUser } from "@/lib/auth-hooks"

export default function UserProfile() {
  const { user, isLoading } = useCurrentUser()
  
  if (isLoading) return <div>Loading...</div>
  return <div>Welcome {user?.first_name}</div>
}
```

### Admin Only:
```tsx
"use client"
import { useRequireAdmin } from "@/lib/auth-hooks"

export default function AdminPage() {
  const { isAdmin, isLoading } = useRequireAdmin()
  
  if (isLoading) return <div>Checking...</div>
  if (!isAdmin) return <div>Access Denied</div>
  return <div>Admin Dashboard</div>
}
```

### Logout Button:
```tsx
"use client"
import { useLogout } from "@/lib/auth-hooks"

export default function Navbar() {
  const logout = useLogout()
  
  return (
    <button onClick={logout}>
      Logout
    </button>
  )
}
```
